
## NAME
ionic link -- Connect your local app to Ionic
  
## SYNOPSIS
    ionic ionic link [app_id]
  
## DESCRIPTION
Connect your local app to Ionic


Input | Description
----- | ----------
`app_id` | The ID of the app to link (e.g. a1b2c3d4)




## EXAMPLES
    ionic ionic link 
    ionic ionic link a1b2c3d4
